◆ ErrorCode

enum Xml::ErrorCode
strong

Error code.

Enumerator
None 

There is no error.

StreamError 

Stream error has occurred.

InvalidByteSequence 

Invalid byte sequence. For example invalid byte order mark (BOM) or alone surrogate halve in the UTF-16 encoding.

UnknownEncoding 

Unknown encoding.

EncodingConfusion 

Encoding confusion. For example UTF-8 from a byte order mark, but UTF-32 in an XML declaration encoding.

EncodingDeclarationRequired 

Encoding declaration must precede content that is not legal UTF-8 or UTF-16.

InvalidSyntax 

Not allowed characters. For example some characters outside a root element, where a white spaces are the only characters allowed.

InvalidXmlDeclarationLocation 

Check http://www.w3.org/TR/2008/REC-xml-20081126/#NT-XMLDecl.

CDataSectionOutside 

CDATA section is outside a root element. Check http://www.w3.org/TR/2008/REC-xml-20081126/#NT-CDSect.

ElementOutside 

Element is outside a root element.

InvalidDocumentTypeDeclarationLocation 

Check http://www.w3.org/TR/2008/REC-xml-20081126/#NT-doctypedecl.

DoubleDocumentTypeDeclaration 

Check http://www.w3.org/TR/2008/REC-xml-20081126/#NT-doctypedecl.

InvalidTagName 

Check http://www.w3.org/TR/2009/REC-xml-names-20091208/#NT-QName.

InvalidAttributeName 

Check http://www.w3.org/TR/2009/REC-xml-names-20091208/#NT-QName.

UnclosedToken 

For example: <tagname attr[end of document].

InvalidReferenceSyntax 

Invalid syntax of the Reference. Check http://www.w3.org/TR/2008/REC-xml-20081126/#NT-Reference.

InvalidCharacterReference 

Code point in the character reference doesn't match the valid character in ISO/IEC 10646 character set. Check http://www.w3.org/TR/2008/REC-xml-20081126/#NT-CharRef.

UnexpectedEndTag 

For example: <a>text</b>. </a> expected, but </b> found. Another example: </b>. Found closing tag, but there is no start tag of b. Both examples are not allowed in the XML files.

UnclosedTag 

For example: <a><b><c></c></b>. Unclosed a tag.

NoElement 

There is no root element in a document.

DoubleAttributeName 

More than one attribute name in the same start-tag or empty-element tag.

PrefixWithoutAssignedNamespace 

There is some name prefix which is not bound to any namespace URI.

PrefixWithEmptyNamespace 

Namespace declaration with a prefix cannot have an empty value.

XmlnsDeclared 

Reserved xmlns prefix cannot be declared or set to an empty value.

PrefixBoundToReservedNamespace 

Prefix is bound to the reserved namespace.

ReservedNamespaceAsDefault 

Reserved namespace cannot be declared as a default namespace.

InvalidXmlPrefixDeclaration 

Prefix 'xml' is reserved for use by the XML and has a fixed namespace URI http://www.w3.org/XML/1998/namespace.

virtual int Xml::Encoding::CharactersReader::ReadCharacter ( char32_t &  result)
pure virtual

Reads one character from the specifically encoded source and translates it to the Unicode character.

The source could be a stream, buffer or anything depending on derived class.

Parameters
[out]resultVariable that will receive the Unicode character.
Returns
1 if the character was read successfully. 0 if there are no more characters to read. -1 if character is not allowed in XML document. -2 if there was an error while reading character.

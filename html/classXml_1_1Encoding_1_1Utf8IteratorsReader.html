Detailed Description

template<typename TInputIterator>
class Xml::Encoding::Utf8IteratorsReader< TInputIterator >

    @brief UTF-8 characters reader from iterators.

    Purpose of this class is an ability to parsing XML documents
    from iterators with a fixed encoding (UTF-8) without
    checking BOM and character set in XML declaration.
    If you are not sure how document is encoded, use Xml::Inspector
    constructor or Xml::Inspector::Reset method with specified
    iterators instead of this reader.

    You can also use the reader object without intermediate
    Xml::Inspector class:
    @code{.cpp}

#include "CharactersReader.hpp" #include <iostream> #include <cstdlib> #include <string>

int main() { // We want to convert UTF-8 string to UTF-32 string. std::string source = u8"<root>Greek small letter pi: \U000003C0.</root>"; std::u32string destination; std::u32string desired = U"<root>Greek small letter pi: \U000003C0.</root>";

Xml::Encoding::Utf8IteratorsReader<std::string::iterator> reader( source.begin(), source.end());

char32_t c; while (reader.ReadCharacter(c) == 1) destination.push_back(c);

if (destination == desired) std::cout << "OK!\n"; else std::cout << "Not OK :(\n";

return EXIT_SUCCESS; } Result: 

        OK!

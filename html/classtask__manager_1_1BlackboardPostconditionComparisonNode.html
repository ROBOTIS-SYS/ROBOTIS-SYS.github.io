Detailed Description

template<typename T>
class task_manager::BlackboardPostconditionComparisonNode< T >

This node excute its child first and check if the value of a given input port is equal to the expected one. If this postcondition is met, this node will return the same status of the child, otherwise it will return the value specified in "return_on_mismatch".

Example:

<ChildBlackboardCheckInt value_A="{child_output}" value_B="42" return_on_mismatch="FAILURE" >

GAEMI0  0.12.0-rc1
  1. 개요
  • 로봇에 설치된 카메라들의 영상을 취득하고, 취득된 영상 상에서 EV 버튼, EV 인디케이터, 특수 (SP) 버튼, 벨 등을 인식하고 포즈를 계산하는 모듈
  • 크게 센서와 디텍터 노드, 이를 관리하는 노드로 구성
  • 센서 노드 : csi_camera, zed, gemini2
  • 디텍터 노드 : ev_button_detector, ev_indicator_detector, sp_button_detector, bell_detector
  • FMS 관련 노드 : fms_pose_calculator
  • 관리자 노드 : perception_manager
  • 관련 파라미터는 aimbot_params/{site}/ai/perception_manager.yaml에서 설정
  • 학습 모델은 colcon_ws/model에서 관리

2. 패키지 설명

  • detector_lib : detector들이 사용하는 inference, pose estimator 알고리즘들의 모음
  • ev_button_detector : EV 내/외부 버튼 인식을 담당
  • ev_indicator_detector : EV 내/외부 인디케이터 인식을 담당
  • sp_button_detector : Tag/door 등 특수 버튼 인식을 담당
  • bell_detector : Bell 버튼 인식을 담당
  • fms_pose_calculator : FMS로부터 bbox 정보를 받고 이를 기반으로 포즈 계산
  • perception_base : 센서와 디텍터들의 상위 클래스를 정의
  • perception_manager : 각 센서와 디텍터 노드들을 관리
  • csi_camera : CSI 카메라 영상 취득을 담당
  • zed : ZED 카메라 영상 취득을 담당
  • gemini2 : Gemini2 영상 취득을 담당

3. 실행

  • aimbot_params/{site}/ai/perception_manager.yaml을 도입할 사이트에 맞게 적절하게 설정
  • colcon_ws/model내 위에서 설정한 모델 유무 확인
  • ros2 launch perception_manager perception_manager.launch.py
